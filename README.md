## Feature

- Second edition algorithm
- Show the calculation results with a sense of ritual
- Multilingual support
- Dark mode

## Developing

```bash
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```

## Building

To create a production version:

```bash
npm run build
```
